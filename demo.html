Flexbox grid

Our flexbox based grid can use any css units, can setting gap and offset, automatic sizing, custom width etc...

Breakpoints

We have six breapoints : xs, sm, md, lg, xl, xxl.
They can be used for responsiveness on any CSS properties, our pseudo-classes, our Felxgrid and for setting classes.

On any screen, width is 100% only from lg screen, width get 55%. 

                          {{brk1}}
